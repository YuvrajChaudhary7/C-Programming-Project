# Media Library Management System (C Program)

The *Media Library Management System* is a simple C program built using basic C concepts such as arrays, structures, file handling, string operations, and menu-driven programming. It allows users to store, view, search, and manage media items such as songs or videos. The program also supports batch import of media records from a CSV file.
